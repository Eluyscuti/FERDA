# Satellite Imagery

Programs to fetch satellite fire data using GOES, MODIS, and VIIRS.

The final server will fetch satellite data, detect fire locations using multispectral imagers, and send this information to the UAV in real-time.

### For context:
    MODIS/VIIRS         - updates every 1-2 days
    GOES                - updates every 10 minutes
    GOES (MESO 1 or 2)  - updates every 1-2 minutes

I have MODIS/VIIRS working but GOES is harder with the quantity of data.
